# Open-source repository studies

ResearchForge stores metadata, notes, and reports about external open-source repositories here.

- Registry metadata lives in each project under `data/oss.json`.
- Local clones belong under `opensource/clones/<owner>/<repo>/` and are ignored by git.
- Do not copy external source code into ResearchForge production code without an explicit license and provenance review.
- Prefer shallow local clones for deterministic study workflows and keep heavyweight refresh jobs optional.
