# Report audit checklist

A complete report should answer:

- What research question was used?
- What sources were searched?
- What exact queries and parameters were used?
- When were searches run?
- Where did each paper come from?
- Which duplicates were merged?
- Why was each paper included or excluded?
- What exact source supports each extracted claim?
- What statistical model was run?
- What inputs, scripts, versions, and outputs reproduce the result?
- What warnings, missing data, or limitations remain?

Golden tests should normalize or isolate timestamps so output stays deterministic.
