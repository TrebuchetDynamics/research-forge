# Fixture policy

Fixture metadata should answer:

- Is this generated, public-domain, open-licensed, or mocked?
- Where did it come from?
- What behavior does it prove?
- Can it be minimized further?
- Does it contain private data, local paths, API keys, copyrighted full text, or secrets?

Preferred fixture directories once implementation starts:

```text
testdata/api/openalex/
testdata/api/arxiv/
testdata/documents/generated/
testdata/grobid/tei/
testdata/projects/
testdata/reports/golden/
```
